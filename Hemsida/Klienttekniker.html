Klienttekniker




HTML

HTML som står för Hypertext Markup Language, är en av webbens grundläggande byggstenar. HTML är själva strukturen för alla internets webbsidor i form av rubriker, styckeindelning, att åtskilja visst innehåll från annat och dylikt. HTML är ett märkspråk och inte ett programmeringsspråk, eftersom HTML-kod endast har i uppdrag att ge struktur.


CSS

CSS som står för Cascading Style Sheets, är en webbens tre grundläggande byggstenar tillsammans med Javascript och HTML. Man kan säga att CSS är webbens ansikte utåt då det sköter allt som har med designen att göra. Med CSS kan man utföra allt från font förändring på text till tuffa animationer.


JavaScript

JS som står för JavaScript är en av webbens tre grundläggande byggstenar med CSS och HTML. Om HTML utgör strukturen och CSS utgör designen kan man säga att JavaScript utgör funktionaliteten. Exempel på JS användande är att någonting förändras när man klickar på en knapp. En vanlig missuppfattning är att JavaScript och Java är samma sak, vilket inte stämmer. Båda är programmeringsspråk, men de är relativt olika i användningen.
